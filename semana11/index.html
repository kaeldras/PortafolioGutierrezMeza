Semana 11: Búsqueda de Artículo Científico – Sistema Inteligente

Descripción de la actividad

En esta semana, realizamos una búsqueda de un artículo científico relacionado con el desarrollo de un sistema inteligente utilizando inteligencia artificial (IA), con el objetivo de seleccionar uno que pudiera servir como base para nuestro proyecto final.

El criterio principal fue que el artículo describiera un sistema implementado en Python, ya que es el lenguaje que estamos utilizando en el curso y que permite una integración directa con tecnologías web.

Después de revisar diversas fuentes científicas (como ScienceDirect, IEEE Xplore y Springer), seleccionamos el siguiente artículo:

Título: Un nuevo enfoque para gestionar datos faltantes para mejorar el sistema de detección de intrusiones en la red
Autores: Mahjabeen Tahir, Azizol Abdullah, Nur Izura Udzir, Khairul Azhar Kasmiran
Revista: Ciberseguridad y aplicaciones (2025)
DOI: 10.1016/j.cyap.2024.100063

Resumen del artículo científico

Este artículo presenta un sistema inteligente de detección de intrusiones en redes (IDS) basado en aprendizaje profundo y gestión de datos faltantes. El sistema, llamado DMDI (Deep Learning-Based Missing Data Imputation), tiene como objetivo mejorar la precisión de los modelos de detección de ciberataques al corregir uno de los principales problemas en los conjuntos de datos de seguridad: los valores faltantes.

Los datos incompletos en conjuntos de datos de redes (como NSL-KDD y UNSW-NB15) pueden afectar negativamente el rendimiento de los modelos de IA. Para resolverlo, los autores proponen un enfoque en dos fases:

  1. Imputación inicial: Usa un codificador automático apilado con reducción de ruido (SDA) para reconstruir los datos faltantes a partir de patrones aprendidos.
  2. Refinamiento: Aplica potenciación de gradiente (Gradient Boosting) para mejorar la precisión de la imputación.

El sistema fue implementado en Python 3, utilizando bibliotecas como Keras, Scikit-learn y NumPy, y probado en una plataforma con Windows 11, procesador Intel Core i5 y 12 GB de RAM.

Resultados clave del sistema

  • Se evaluó con cinco clasificadores: SVM, KNN, regresión logística, árbol de decisión y bosque aleatorio.
  • Después de la imputación con DMDI, se lograron mejoras de precisión entre 0.95 y 0.97.
  • El método demostró ser más eficiente que técnicas tradicionales como KNN, MICE y regresión múltiple.
  • El sistema es especialmente útil en entornos donde la calidad de los datos es crítica: ciberseguridad, salud, finanzas y redes de sensores.

Relevancia para el proyecto final

Este artículo es altamente relevante para nuestro proyecto final porque:

  • Es un ejemplo real de un sistema inteligente desarrollado en Python.
  • Combina IA, preprocesamiento de datos y análisis de seguridad, temas que podemos adaptar.
  • La metodología es clara, replicable y bien documentada.
  • Podemos usar partes del código (como el autoencoder o los clasificadores) como base para nuestro propio sistema.
  • Demuestra cómo la calidad de los datos influye directamente en el rendimiento del modelo, un aspecto clave en cualquier proyecto de IA.
