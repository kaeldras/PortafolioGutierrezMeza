Descripción de temas aprendidos (Exposiciones Grupales)

En esta semana, el aprendizaje se consolidó a través de exposiciones grupales, donde cada equipo profundizó en temas específicos de las tecnologías web fundamentales. Esto permitió una visión más detallada y colaborativa sobre los siguientes pilares del desarrollo Front-End:

HTML Básico y Avanzado

  • **HTML Básico:** Revisamos los fundamentos de la estructura de una página web, las etiquetas esenciales (`<html>`, `<head>`, `<body>`, `<h1>`-`<h6>`, `<p>`, `<a>`, `<img>`, listas, etc.) y cómo estas forman el esqueleto del contenido.
  • **HTML Avanzado:** Exploramos elementos más complejos y semánticos de HTML5, como etiquetas de sección (`<header>`, `<nav>`, `<main>`, `<article>`, `<aside>`, `<footer>`, `<section>`), formularios avanzados y elementos multimedia, enfocándonos en la estructuración de contenido accesible y compatible con SEO.

CSS Básico y Avanzado, y CSS con Frameworks

  • **CSS Básico:** Se repasaron los conceptos de estilos, selectores (clase, ID, etiqueta), el Modelo de Caja (Box Model), y la aplicación de propiedades básicas como color, fondo, tipografía y espaciado.
  • **CSS Avanzado:** Nos adentramos en técnicas de diseño más sofisticadas, incluyendo transformaciones, transiciones, animaciones y pseudoclases/pseudoelementos. Se resaltó la importancia de las "Media Queries" para el diseño responsivo.
  • **CSS con Bootstrap:** Estudiamos Bootstrap, uno de los frameworks de CSS más populares, que facilita la creación de interfaces responsivas y modernas mediante su sistema de grillas y componentes predefinidos. Se abordó cómo usar sus clases utilitarias y personalizar el diseño.
  • **CSS con Tailwind:** Se presentó Tailwind CSS, un framework utilitario que permite construir diseños personalizados rápidamente aplicando clases de bajo nivel directamente en el HTML, promoviendo un enfoque de "Atomic CSS" para una mayor flexibilidad y rendimiento. Se discutieron sus ventajas en el diseño responsivo y la personalización.

JavaScript Básico

  • **Introducción a JavaScript:** Reafirmamos su rol como lenguaje de programación del lado del cliente, esencial para añadir interactividad y dinamismo a las páginas web.
  • **Fundamentos de Programación:** Se abordaron variables, tipos de datos (Strings, Numbers, Booleans, Arrays, Objects), operadores (aritméticos, lógicos, comparación), estructuras condicionales (`if`, `else`, `switch`) y bucles (`for`, `while`, `do while`).
  • **Funciones y Manipulación del DOM:** Se cubrió la declaración y uso de funciones para modular el código, así como las técnicas básicas para manipular el Document Object Model (DOM), permitiendo seleccionar elementos, modificar su contenido y responder a eventos de usuario (`addEventListener`, `onclick`).

Las exposiciones enriquecieron nuestro conocimiento sobre cada tecnología y cómo se complementan para construir aplicaciones web robustas y amigables con el usuario.

Ejercicios de laboratorio - Resultados (PRACTICA02)
