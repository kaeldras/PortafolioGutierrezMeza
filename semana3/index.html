Semana 3: Introducción a JavaScript (ECMAScript)

Descripción de temas aprendidos

La Semana 3 marcó la entrada al mundo de la **programación dinámica** del lado del cliente con **JavaScript**. Aprendimos que JavaScript es un lenguaje de programación script dinámico, multi-paradigma, basado en prototipos, que soporta estilos de programación funcional, orientada a objetos e imperativa.

Se destacó su evolución, especialmente con el lanzamiento de Node.js, lo que lo ha convertido en un lenguaje fundamental no solo para la web, sino también para desarrollar:

  • Aplicaciones de servidor.
  • Aplicaciones móviles.
  • Aplicaciones de escritorio.
  • Aplicaciones de consola.
  • Aplicaciones de robótica.
  • Aplicaciones empotradas (SmartTV).
  • Por supuesto, aplicaciones web.

Se mencionó que JavaScript es el lenguaje de programación más popular, aunque curiosamente, es un lenguaje que muchos desarrolladores aprenden en el trabajo más que en la universidad.

Fundamentos de JavaScript (ECMAScript)

El estándar de JavaScript es **ECMAScript**. Desde 2012, todas las versiones de JavaScript siguen este estándar. Abordamos los conceptos básicos para empezar a programar:

  • **Variables y Tipos de Datos:** Cómo declarar variables y los tipos de datos principales en JavaScript (números, cadenas de texto, booleanos, `null`, `undefined`, objetos).
  • **Operadores:** Los diferentes tipos de operadores para realizar operaciones aritméticas, de comparación, lógicas, de asignación, entre otros.
  • **Estructuras de Control:** La base para la lógica de los programas:
    • **Condicionales:** `if`, `else if`, `else` para tomar decisiones.
    • **Bucles (Loops):** `for`, `while`, `do...while` para repetir bloques de código.
    • **Estructuras de control de selección:** `switch` para múltiples opciones.
  • **Funciones:** Cómo definir y llamar funciones para organizar el código en bloques reutilizables.
  • **Programación Orientada a Objetos (POO):** Una introducción a los conceptos de objetos, propiedades y métodos.

Además, se explicó la diferencia entre **JavaScript (el lenguaje de programación)** y **Java (un lenguaje de programación completamente diferente)**, un punto crucial para evitar confusiones comunes.

Esta semana sentó las bases para el desarrollo de funcionalidades interactivas en las páginas web, pasando de la estructura y el estilo a la lógica y el comportamiento dinámico.
