Semana 1: Introducción a la Tecnología Web y Herramientas de Desarrollo

Descripción de temas aprendidos

Durante esta primera semana del curso de Desarrollo de Aplicaciones Web, nos introdujimos a los **conceptos fundamentales de la tecnología web**. Se discutió la importancia de tener un portafolio electrónico, como este mismo, para documentar nuestro aprendizaje de forma continua y accesible.

Uno de los puntos centrales fue la familiarización con **Visual Studio Code (VSC)**, una herramienta esencial para cualquier desarrollador web. Aprendimos sobre:

  • La descarga e instalación de VSC.
  • La interfaz de usuario principal de VSC y sus áreas clave.
  • El uso de la Paleta de Comandos para ejecutar diversas funciones.
  • La Terminal integrada de VSC, útil para comandos de Git.
  • La funcionalidad de **Emmet**, que permite generar código HTML y CSS de forma rápida y eficiente mediante abreviaciones.
  • Conceptos básicos de Git como git init, git add, git commit, git checkout, git push, ramas y *merge* en GitHub, aunque sin ejercicios prácticos esta semana.

También se abordaron temas introductorios sobre la **evolución de la Web (Web 1.0, 2.0, 3.0)**, y herramientas de diseño y prototipado como **Figma**, destacando su colaboración en tiempo real.

Ejercicios de laboratorio - Resultados

En esta semana, el enfoque fue principalmente teórico y de familiarización con las herramientas. No se realizaron ejercicios de laboratorio con resultados a mostrar.

Reflexión: ¿Qué aprendí? y ¿Cómo aprendí?

La Semana 1 fue una base sólida para entender el entorno de desarrollo web. Aprendí la relevancia de un portafolio electrónico y la versatilidad de Visual Studio Code como mi IDE principal. La introducción a Emmet fue particularmente interesante por su potencial para agilizar el desarrollo. Aunque no hubo ejercicios prácticos, la exposición a los comandos básicos de Git y a la filosofía de la Web 2.0 y 3.0 sentó las bases para lo que vendrá. Mi aprendizaje se dio a través de la exploración guiada de VSC y la asimilación de conceptos teóricos fundamentales para el desarrollo web.
