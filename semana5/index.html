Semana 5: Introducción a React y Desarrollo de Aplicaciones con React

Descripción de temas aprendidos

En esta semana, dimos un salto crucial en el desarrollo web, adentrándonos en el mundo de **React**, una de las librerías de JavaScript más populares y potentes para construir interfaces de usuario interactivas y dinámicas.

Los temas centrales cubiertos incluyeron:

  • **¿Qué es React?** Comprendimos que React no es un framework, sino una librería de JavaScript enfocada en la capa de vista de las aplicaciones, ideal para construir Single Page Applications (SPAs).
  • **Historia de React.js:** Revisamos su origen en Facebook y cómo ha evolucionado para convertirse en una herramienta indispensable en el ecosistema de desarrollo frontend.
  • **Cómo funciona React:** Exploramos el concepto fundamental del **Virtual DOM**, que permite a React optimizar las actualizaciones de la interfaz de usuario, renderizando solo los componentes que han cambiado para mejorar el rendimiento.
  • **Componentes en React:** Aprendimos la base de React: los componentes. Entendimos cómo estos bloques de construcción reutilizables y aislados nos permiten crear interfaces complejas de manera modular y escalable.
  • **JSX (JavaScript XML):** Nos familiarizamos con JSX, una extensión de sintaxis que permite escribir HTML dentro de JavaScript, facilitando la creación y visualización de la estructura de los componentes.
  • **Props y State:** Diferenciamos entre `props` (propiedades, datos que se pasan de un componente padre a un hijo, son inmutables) y `state` (estado, datos internos de un componente que pueden cambiar y provocar un re-renderizado del componente).
  • **Ciclo de Vida de los Componentes (introducción):** Obtuvimos una primera aproximación a las fases por las que pasa un componente (montaje, actualización y desmontaje) y cómo podemos ejecutar código en cada una de ellas.
  • **Manejo de Eventos:** Aprendimos a gestionar las interacciones del usuario (clics, cambios en formularios, etc.) en React, conectando funciones a los eventos de los elementos del DOM.
  • **Creación de un Proyecto en React:** Introducción a las herramientas para iniciar proyectos React, como Create React App y Vite, comprendiendo sus diferencias y ventajas.
  • **Estructura de un Proyecto en React:** Analizamos la organización de archivos y carpetas típica en un proyecto React, esencial para mantener el código ordenado y manejable.

Esta semana sentó las bases para el desarrollo de aplicaciones web modernas, interactuando con conceptos que nos permitirán crear interfaces de usuario más robustas y eficientes.

Ejercicios de laboratorio - Resultados

Durante la práctica de laboratorio de esta semana, aplicamos los conceptos de React para desarrollar componentes básicos y funcionalidades interactivas. Esto incluyó la creación de un componente principal que mostraba información y la manipulación del estado para cambiar elementos en la interfaz. También, exploramos cómo pasar datos entre componentes utilizando `props` para hacerlos más reutilizables y configurables.
