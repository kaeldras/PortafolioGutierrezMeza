Semana 2: Fundamentos de HTML, CSS y Productividad con Emmet

Descripción de temas aprendidos

En esta segunda semana, profundizamos en los lenguajes fundamentales para el desarrollo web: **HTML (HyperText Markup Language)** y **CSS (Cascading Style Sheets)**.

Aprendimos cómo HTML estructura el contenido de una página web a través de sus diversas etiquetas (como '

', '

', '', '

', '', etc.) y cómo se construye el **Árbol DOM (Document Object Model)**.

En cuanto a CSS, exploramos cómo dar estilo a ese contenido HTML. Se cubrieron temas esenciales como:

  • **¿Qué es CSS?** Y su importancia para la presentación visual.
  • Las diferentes formas de **incluir CSS** en un documento HTML (inline, interno y externo).
  • Los **selectores CSS** (etiqueta, clase, ID) y su uso para aplicar estilos a elementos específicos.
  • **Pseudo-clases** (como `:hover`, `:active`) para estilos basados en el estado del elemento.
  • El **Box Model (Modelo de Caja)**, que explica cómo se distribuyen los elementos en la página (margen, borde, relleno, contenido).
  • Unidades de longitud (absolutas y relativas).
  • Introducción a **Flexbox y Grid Layout** como herramientas avanzadas para la maquetación responsiva.
  • Conceptos de **Diseño Responsivo**, **Inclusión y Accesibilidad Universal** (WCAG 2.1), y las propiedades ARIA.

Además, reforzamos el uso de **Emmet**, una herramienta de productividad integrada en Visual Studio Code que permite generar estructuras HTML y CSS complejas de forma muy rápida utilizando abreviaciones. Esto optimiza significativamente el tiempo de desarrollo.

Ejercicios de laboratorio - Resultados

Como ejercicio principal de la semana, trabajamos en la generación de código HTML utilizando abreviaciones de **Emmet**. Esto nos permitió practicar la sintaxis y ver la eficiencia de esta herramienta para construir rápidamente elementos comunes de una página web, como menús de navegación, tablas y estructuras semánticas.

El resultado de esta práctica incluyó la generación de:

  • Un menú de navegación (`<nav>`).
  • Tablas con `thead` y diversas filas.
  • Estructuras HTML semánticas (`<header>`, `<nav>`, `<section>`, `<article>`, `<aside>`, `<footer>`).
  • Tablas más grandes con estructuras específicas.

Este ejercicio práctico consolidó nuestro entendimiento de cómo Emmet acelera el proceso de escritura de código.
